NOMBRE DEL PROYECTO:
Sistema de Inventario de Tienda 

DESCRIPCIÓN:
Sistema para gestionar productos de una tienda, permitiendo registrar, consultar y visualizar inventario.

FUNCIONALIDADES:
- Registrar productos
- Consultar productos
- Editar productos
- Ver inventario

CÓMO EJECUTAR:
1. Descargar el proyecto o abrirlo en Visual Studio Code
2. Abrir el archivo index.html en el navegador

TECNOLOGÍAS:
- HTML
- CSS
- JavaScript
